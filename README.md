This repository contains hands-on workshop materials for CS-GY-9223  Designing XR Environments for Skill Learning at NYU Tandon School of Engineering, Spring 2026. 
Across four workshops, students learn to build immersive VR applications in Unity 2022.3 for Meta Quest 3 — starting from project setup and grab interactions, through locomotion, multimodal instruction systems (text, video, spatial audio, animation, haptics), multiplayer networking with Photon PUN2, and AI-powered pedagogical agents. 
Each workshop is a live-coding session where students follow along and build directly on a running Unity project, learning by doing rather than by reading.

Workshop 1 — XR Foundations, Unity Setup and locomotion
Install Unity and set up their first XR project from scratch, learning how to configure the XR Interaction Toolkit, build grab and trigger interactions, and test in XR Device Simulator.Implementing continuous movement and teleportation so players can actually navigate the VR environment.
 
Workshop 2 — Multimodal Instruction in XR
Extend the VR project with five feedback modalities world-space text, in-world video playback, 2D and spatial audio, coroutine-driven animation, and three types of haptic feedback — learning how to guide users through tasks without relying on text-heavy interfaces.

Workshop 3 — Multiplayer VR
Build a shared VR room from scratch using Photon PUN2, where multiple players appear as networked avatars with visible heads and hands, can spawn and transfer ownership of objects, and interact with the same environment simultaneously.

Workshop 4 — AI in XR Learning Environments
Integrate AI-powered NPCs into the VR scene, virtual tutors that detect when a learner approaches and respond in real time using OpenAI, with two interaction modes: a text-based NPC using an in-headset keyboard, and a voice-based NPC that accepts spoken input ,covering both input modalities and the gap AI fills in pre-authored training environments.
