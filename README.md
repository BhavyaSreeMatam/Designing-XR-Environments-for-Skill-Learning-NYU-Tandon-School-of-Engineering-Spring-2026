This repository contains hands-on workshop materials for CS-GY-9223  Designing XR Environments for Skill Learning at NYU Tandon School of Engineering, Spring 2026. 
Across four workshops, students learn to build immersive VR applications in Unity 2022.3 for Meta Quest 3 — starting from project setup and grab interactions, through locomotion, multimodal instruction systems (text, video, spatial audio, animation, haptics), multiplayer networking with Photon PUN2, and AI-powered pedagogical agents. 
Each workshop is a live-coding session where students follow along and build directly on a running Unity project, learning by doing rather than by reading.
